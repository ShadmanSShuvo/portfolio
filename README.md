### Using My Portfolio Website as Template
- Fork this repository.
- Clone your forked repository.
- Install dependencies.
```
npm install
```

- Run in local machine.
```
npm run dev
```

- Update the static data files in ```src/lib/data``` with your data.
- Voila!
